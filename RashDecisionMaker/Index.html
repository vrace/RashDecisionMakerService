Welcome to use Rash Decision Maker RESTful Service

Available APIs

/d[/upper bound]
Generates a random number from 1 ~ upper bound.
Example:
http://some_end_point/d/100
Response json:
{ "upper": "100", "result": "42" }

/pick/<a|b|c>
Pick an item from a, b, c
Example:
http://some_end_point/pick/a|b|c
Response json:
{ "item": "c" }

/pair/<a|b|c>
Pair a collection of items
Example:
http://some_end_point/pair/one|two|three
Response json:
{ "numPairs": "2", "pair0": { "a": "one", "b": "two" }, "pair1": { "a": "three"} }
